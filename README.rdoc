Проект RubyBursa:

Автоматизация досуга
Учет прочитанных книг, просмотренных фильмов, посещенных ресторанов и т.д.
Добавление мест, ресурсов "на будущее" с ссылками (где о них можно почитать)
Добавление отзывов, комментариев, возможно, цитат из книжек и т.д.
